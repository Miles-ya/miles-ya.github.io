关于

本博客以 Obsidian 进行内容创作，使用 Hugo 生成静态站点，并通过 GitHub Actions 自动构建与部署到 GitHub Pages，便于在本地用 Obsidian 管理笔记，同时实现自动化发布与更新。

一点想法

拥有一个博客是我一直以来渴望做的事。 之前我也在腾讯云上使用 Wordpress 建过一个网站，但是因为云主机的续费费用超出了我这个穷学生的预算，便不了了之。 Github pages 我也折腾了好几次，从 Hexo 到现在的 Hugo 始终直到现在才成功跑通从写作到发布的整个链条。

至于为什么要建站，还是想有一个可以自由表达的地方，可以不为了流量和金钱，只为自由自在地表达。

作为一个工科生，我也并没有深厚的文学素养，但是只要开始写就足够了。

我很享受在写作时的快感，尤其是在一个偌大的房间，只剩我一个人的时候，我感觉不到时间的流逝，只有表达带给我的喜悦。

致青春！

技术栈

  • Obsidian - Markdown 编辑器，用于内容创作和管理
  • Obsidian-Hugo-Sync - Obsidian 发布插件（我二次开发的，原项目地址Obsidian-Hugo-Sync）
  • Hugo - 静态网站生成器
  • Hello Friend NG - Hugo 主题
  • GitHub Pages - 网站托管
  • GitHub Actions - 自动化部署
